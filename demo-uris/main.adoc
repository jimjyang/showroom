== The purpose [[purpose]]

This showroom is meant to

* demonstrate a possible pattern of URIs, in accordance with the Norwegian URI-standard;
* demonstrate a possible way to realize permanent and dereferenced URIs.

In this showroom we demonstrate how to use the service for permanent URIs provided by https://w3id.org[w3id.org].
